# 快速启动本地开发环境

## 1. 进入项目目录（重要！）
cd /Users/renzhiliao/Desktop/learning-platform

## 2. 安装依赖（首次运行或更新后需要）
npm install

## 3. 启动前端开发服务器
npm run dev

## 4. 新开一个终端窗口，启动后端服务器
# 先进入项目目录
cd /Users/renzhiliao/Desktop/learning-platform
# 启动后端
npm run start

## 访问地址
# 前端: http://localhost:5173
# 后端API: http://localhost:3002

pwd          # 查看当前目录
