# Navicat SSH隧道连接配置

## 通过SSH隧道连接阿里云RDS

### 1. 在Navicat中配置SSH隧道

#### 常规设置：
- **连接名称**：阿里云RDS（通过SSH）
- **主机**：rm-cn-7js4el1by00015fo.rwlb.cn-chengdu.rds.aliyuncs.com
- **端口**：3306
- **用户名**：admin123
- **密码**：Liao0820

#### SSH设置（重要）：
点击【SSH】标签页，配置：
- ✅ 勾选【使用SSH隧道】
- **主机**：47.109.142.72（你的宝塔服务器）
- **端口**：22
- **用户名**：root
- **验证方法**：密码
- **密码**：（你的服务器root密码）

### 2. 工作原理
```
你的电脑 → SSH连接 → 宝塔服务器(47.109.142.72) → 阿里云RDS
```

这样就绕过了IP白名单限制，因为是从你的服务器（已在白名单中）访问RDS。

### 3. 测试连接
配置完成后，点击【测试连接】，应该会成功。

### 4. 如果SSH也连接失败
可能需要在服务器上开放SSH端口：
- 登录宝塔面板
- 【安全】→ 确保22端口已放行
- 阿里云安全组也要放行22端口
